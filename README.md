# Portfolio Landing

Адаптивный лендинг-портфолио на чистом `HTML`, `CSS` и `JavaScript` с темной и светлой темой, анимациями.

## Что внутри

- современный одностраничный лендинг
- адаптивная верстка для desktop, tablet и mobile
- переключение светлой и темной темы
- анимированный hero-блок
- секция с опытом, стеком и контактами

## Стек

- `HTML5`
- `CSS3`
- `JavaScript`

## Структура

```text
.
├── index.html
├── styles.css
└── script.js
```

## Как запустить

### Вариант 1. Просто открыть

Откройте `index.html` в браузере.

### Вариант 2. Через локальный сервер

Если хотите запускать проект как обычный локальный сайт, можно поднять простой сервер.

#### Python

```bash
python -m http.server 4173
```

После этого откройте:

```text
http://localhost:4173
```

## Что можно менять

- тексты и блоки: `index.html`
- стили и адаптив: `styles.css`
- тема и анимации: `script.js`
